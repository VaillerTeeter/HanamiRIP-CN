# Security Policy

## Supported Versions

目前仅维护最新主分支（`master`/`main`）的安全修复与公告。

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| Older  | ❌ |

## Reporting a Vulnerability

请通过以下方式提交安全问题（请勿公开发布细节）：

- 方式 1：使用 GitHub 私有安全报告（Security Advisories）
- 方式 2：发送邮件至：wyc_19533480830@outlook.com

请在报告中提供：

- 复现步骤（尽量最小化）
- 受影响的版本/提交
- 预期结果与实际结果
- 可能的修复建议（可选）

我们会在收到报告后的 7 个工作日内回复，并在评估后尽快修复与发布。
